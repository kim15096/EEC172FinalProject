# Setting up Webserver

Webserver using websockets and AWS IOT sdk and webrtc sdk.

## Getting Started

### Installing

```
npm install
```

### ENV variables

Create a .env file in the webserver directory with your AWS credentials.
```
AWS_ACCESS_KEY_ID=""
AWS_SECRET_ACCESS_KEY=""
AWS_DEFAULT_REGION=""
```

### Executing program

```
node server.js
```
