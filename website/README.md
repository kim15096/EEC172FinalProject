# Setting up Website

LazerCat frontend website to view live stream and select lazer modes.

## Getting Started

### Installing

```
npm install
```

### ENV variables
```
VITE_AWS_ACCESS_KEY=""
VITE_AWS_SECRET_KEY="+"
VITE_AWS_HOST_NAME=""
VITE_AWS_REGION=""
VITE_AWS_SIGNAL_ARN=""
VITE_SERVER_URL=""
VITE_SERVER_SOCKET=""
```

### Executing program
This serves the website at http://localhost:5173

```
npm run dev
```
